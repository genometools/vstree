\begin{minipage}[t]{0.75\linewidth}
This is the documentation of general library functions compiled into
the archive \texttt{libkurtz.a}. The include file used for compiling this
library can be found in directory \texttt{include} and the library files
in directory \texttt{kurtz-basic} and \texttt{kurtz}.
For each library file, say \texttt{libfile.c} there exists a corresponding
file \texttt{include/libfile.pr} containing the header of the functions
exported from \texttt{libfile.c}.
The prototypes of all library functions can be found in this document.
The PDF-version of this document comes with hyperlinks,
linking type names, function names, and macro names to their definition.
The acrobat reader \texttt{acroread} conveniently allows navigating along
these links. If you encounter broken or missing links, please contact the
author at \texttt{kurtz@zbh.uni-hamburg.de}. Other comments
are also welcome.
\end{minipage}
